Usage Tips
General tips:
  • A "forward swipe" will jump you straight into the remote
  • There are global shortcuts for the other app scenes too
  • The app has search built into most scenes - just start typing to search
  • Tapping the header will usually make the list jump to the top
Remote tips:
  • The keyboard also works for sending commands to the frontend. This is great for some more advanced remote usage
  • Tapping the header on a remote scene will pause or mute (selected in the preferences)
